# People & Prompt Labs

Static browser-based utilities published under the People & Prompt Labs site.

## Utilities

- `Invida-Upload-Validator/` - Excel workbook validation tool for Utility / Invida bulk uploads.

## Privacy

These tools are static HTML, CSS, and JavaScript. The Invida upload validator reads Excel files locally in the user's browser and does not upload workbook contents to a server.

## Deployment

This folder is intended to be deployed as the static site root for `labs.peopleandprompt.com`.
